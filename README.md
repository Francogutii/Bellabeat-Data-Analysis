# Bellabeat-Data-Analysis
Análisis de datos sobre hábitos de sueño y actividad física para la empresa Bellabeat (Google Data Analytics Capstone).

# Caso de Estudio Bellabeat: Análisis de Actividad y Sueño

## 1. Introducción
Este proyecto forma parte del Certificado de Análisis de Datos de Google. El objetivo fue analizar datos de dispositivos inteligentes para descubrir tendencias que ayuden a la estrategia de marketing de Bellabeat.

## 2. Herramientas Utilizadas
* **Google Sheets:** Limpieza de datos, tablas dinámicas y cálculos (VLOOKUP).
* **Análisis Estadístico:** Regresión lineal simple.

## 3. Hallazgo Principal: ¿El sedentarismo afecta el sueño?
Contrario a la creencia popular, mi análisis descubrió que **no existe una correlación fuerte** entre el tiempo que un usuario pasa sentado y la cantidad de minutos que duerme.

![Gráfico de Sedentarismo](AQUI_IRÁ_TU_GRAFICO)

*Como se observa en el gráfico, la tendencia es casi plana (Correlación negativa débil), lo que sugiere que el descanso depende de otros factores.*

## 4. Conclusión y Recomendaciones
Basado en los datos, recomiendo a Bellabeat no enfocar su marketing únicamente en "caminar más para dormir mejor", sino explorar funciones de relajación y reducción de estrés.
